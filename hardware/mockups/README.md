# Mockups

Concept renders of the finished build, for reference while the OpenSCAD case is developed.

- `mockup-open.svg` — open clamshell: lid with two IPS displays, 60% keyboard, embedded T-Embed CC1101 module bay, front-edge ports
- `mockup-closed.svg` — folded travel state with approximate dimensions and the USB-C PD charge input

These are illustrative, not to scale. Dimensions firm up once the panels, keyboard and T-Embed board are sourced and measured.
