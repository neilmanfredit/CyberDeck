# Mockups

Concept renders of the finished build, for reference while the OpenSCAD case is developed.

- `mockup-open.svg` / `mockup-open.png` — open clamshell: lid with two IPS displays, 60% keyboard, embedded T-Embed CC1101 module bay, front-edge ports
- `mockup-closed.svg` / `mockup-closed.png` — folded travel state with approximate dimensions and the USB-C PD charge input

SVGs are the source; PNGs are provided because GitHub renders them reliably in the README.

These are illustrative, not to scale. Dimensions firm up once the panels, keyboard and T-Embed board are sourced and measured.
